# Spec-002: 全局UI设计系统

## 1. 功能概述

定义项目的全局设计规范，包括颜色体系、字体规范、间距系统、组件样式、布局模板等，确保整个应用的视觉风格统一，呈现简约科技感。

---

## 2. 技术选型

| 技术 | 用途 |
|------|------|
| TailwindCSS | 原子化CSS，实现设计系统 |
| shadcn/ui | 基础组件库 |
| Framer Motion | 动效库 |
| next/font | 字体优化 |
| CSS Variables | 主题变量 |
| Lucide React | 图标库 |

---

## 3. 设计规范

### 3.1 设计理念

- **简约**：去除多余装饰，聚焦内容本身
- **科技**：使用渐变、光效、微动效体现科技感
- **专业**：清晰的层次结构，舒适的视觉体验
- **现代**：大圆角、柔和阴影、充足留白

```tsx
interface Step {

  steps: Step[]
  className?: string

export function StepIndicator({ steps, className }: StepIndicatorProps) {
  return (
    <div className={cn("flex items-center justify-center", className)}>
      {steps.map((step, index) => (
        <div key={step.id} className="flex items-center">
          {/* 步骤圆圈 */}
          <div className="flex flex-col items-center">
            <div
              className={cn(
                "flex h-10 w-10 items-center justify-center rounded-full border-2 transition-all duration-300",
                step.status === 'completed' && "border-accent-500 bg-accent-500 text-white",
                step.status === 'current' && "border-primary-500 bg-primary-500 text-white shadow-glow-primary",
                step.status === 'upcoming' && "border-neutral-200 bg-white text-neutral-400"
              )}
            >
              {step.status === 'completed' ? (
                <Check className="h-5 w-5" />
              ) : (
                <span className="text-sm font-medium">{index + 1}</span>
              )}
            </div>
            <span
              className={cn(
                "mt-2 text-xs font-medium",
                step.status === 'completed' && "text-accent-600",
                step.status === 'current' && "text-primary-600",
                step.status === 'upcoming' && "text-neutral-400"
              )}
            >
              {step.label}
            </span>
          </div>
          
          {/* 连接线 */}
          {index < steps.length - 1 && (
            <div
              className={cn(
                "mx-2 h-0.5 w-12 sm:w-20 transition-all duration-300",
                step.status === 'completed' ? "bg-accent-500" : "bg-neutral-200"
              )}
            />
          )}

  )
```

### 4.6 进度条组件

```tsx
// frontend/src/components/shared/progress-bar.tsx

import { cn } from "@/lib/utils"

interface ProgressBarProps {
  value: number // 0-100
  label?: string
  showValue?: boolean
  className?: string
  variant?: 'default' | 'success' | 'gradient'
}

export function ProgressBar({ 
  value, 
  label, 
  showValue = true, 
  className,
  variant = 'default'
}: ProgressBarProps) {
  const clampedValue = Math.min(100, Math.max(0, value))
  
  const variantClasses = {
    default: 'bg-primary-500',
    success: 'bg-accent-500',
    gradient: 'bg-gradient-to-r from-primary-500 to-accent-500',
  }

  return (
    <div className={cn("w-full", className)}>
      {(label || showValue) && (
        <div className="mb-2 flex items-center justify-between">
          {label && <span className="text-sm font-medium text-neutral-700">{label}</span>}
          {showValue && (
            <span className="text-sm font-medium text-neutral-500">{clampedValue}%</span>
          )}
        </div>
      )}
      <div className="h-2 w-full overflow-hidden rounded-full bg-neutral-100">
        <div
          className={cn(
            "h-full rounded-full transition-all duration-500 ease-out",
            variantClasses[variant]
          )}
          style={{ width: `${clampedValue}%` }}
        />
      </div>
    </div>
  )
}
```

```tsx
          )}

  )
```

## 7. 动效规范

### 7.1 过渡时间

| Token | 值 | 用途 |
|-------|------|------|
| duration-instant | 100ms | 微交互（hover等） |
| duration-fast | 150ms | 按钮状态变化 |
| duration-normal | 200ms | 默认过渡 |
| duration-slow | 300ms | 面板展开 |
| duration-slower | 500ms | 页面过渡 |

### 7.2 缓动函数

```css

--ease-default: cubic-bezier(0.4, 0, 0.2, 1);

/* 退出 */
--ease-out: cubic-bezier(0, 0, 0.2, 1);

--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 7.3 页面过渡动画

```tsx
// frontend/src/components/shared/page-transition.tsx

"use client"

import { motion, AnimatePresence } from "framer-motion"
import { usePathname } from "next/navigation"

const pageVariants = {

    y: 10,
  },
  animate: {
    opacity: 1,
    y: 0,

      duration: 0.3,
      ease: [0.4, 0, 0.2, 1],
    },
  },

    y: -10,
    transition: {
      duration: 0.2,
    },
  },
}

      variants={pageVariants}
      initial="initial"
      animate="animate"
      exit="exit"
    >
      {children}
    </motion.div>
  )
}

// 列表项动画
export function ListItemTransition({ 
  children, 
  index = 0 
}: {

  index?: number

      initial={{ opacity: 0, y: 10 }}
      animate={{ opacity: 1, y: 0 }}
      transition={{
        duration: 0.3,
        delay: index * 0.05,
        ease: [0.4, 0, 0.2, 1],
      }}
    >
      {children}
    </motion.div>
  )
```
