# Spec-008: 大纲配置页面

## 1. 功能概述

用户选择标题后，进入大纲配置页面。本页面让用户填写大纲生成的相关需求参数：
- Word Count（文章字数）
- Readability（可读性/阅读难度）
- Tone of Voice（语调/写作风格）
- Point of View（人称视角）
- Additional Requirements（额外要求）

这些参数将传递给LLM用于生成符合用户需求的大纲。

## 3. 数据模型

### 3.1 大纲配置表

```sql
CREATE TABLE outline_configs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    project_id UUID UNIQUE NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
    
    -- 配置参数
    word_count INTEGER NOT NULL DEFAULT 2000,
    readability VARCHAR(50) NOT NULL DEFAULT 'general',
    tone_of_voice VARCHAR(50) NOT NULL DEFAULT 'professional',
    point_of_view VARCHAR(50) NOT NULL DEFAULT 'third_person',
    additional_requirements TEXT,
    
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

## 4. API设计

### 4.1 保存大纲配置

```
POST /api/projects/{project_id}/outline/config
```

**Request Body:**
```json
{
  "word_count": 2000,
  "readability": "general",
  "tone_of_voice": "professional",
  "point_of_view": "third_person",
  "additional_requirements": "请重点关注2024年的最新发展"

```

**Response 200:**

{

### 4.2 后端实现

```python
# backend/app/api/endpoints/outline.py

from fastapi import APIRouter, Depends, HTTPException
from sqlalchemy.ext.asyncio import AsyncSession
from pydantic import BaseModel, Field
from typing import Optional

from app.database import get_db
from app.models.outline import OutlineConfig
from app.models.project import Project

router = APIRouter()

class OutlineConfigRequest(BaseModel):
    word_count: int = Field(default=2000, ge=500, le=10000)
    readability: str = Field(default="general")
    tone_of_voice: str = Field(default="professional")
    point_of_view: str = Field(default="third_person")
    additional_requirements: Optional[str] = Field(default=None, max_length=2000)

@router.post("/projects/{project_id}/outline/config")
async def save_outline_config(
    project_id: str,
    config: OutlineConfigRequest,
    db: AsyncSession = Depends(get_db)
):
    """保存大纲配置"""
    project = await db.get(Project, project_id)
    if not project:
        raise HTTPException(status_code=404, detail="Project not found")
    
    # 获取或创建配置
    existing = await db.execute(
        select(OutlineConfig).where(OutlineConfig.project_id == project_id)
    )
    outline_config = existing.scalar_one_or_none()
    
    if outline_config:
        # 更新
        outline_config.word_count = config.word_count
        outline_config.readability = config.readability
        outline_config.tone_of_voice = config.tone_of_voice
        outline_config.point_of_view = config.point_of_view
        outline_config.additional_requirements = config.additional_requirements
    else:
        # 新建
        outline_config = OutlineConfig(
            project_id=project_id,
            word_count=config.word_count,
            readability=config.readability,
            tone_of_voice=config.tone_of_voice,
            point_of_view=config.point_of_view,
            additional_requirements=config.additional_requirements
        )
        db.add(outline_config)
    
    # 更新项目状态
    project.status = "outline_config_saved"
    
    await db.commit()
    await db.refresh(outline_config)
    
    return {
        "id": str(outline_config.id),
        "project_id": str(outline_config.project_id),
        "word_count": outline_config.word_count,
        "readability": outline_config.readability,
        "tone_of_voice": outline_config.tone_of_voice,
        "point_of_view": outline_config.point_of_view,
        "additional_requirements": outline_config.additional_requirements,
    }
```
