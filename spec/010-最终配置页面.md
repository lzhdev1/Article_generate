# Spec-010: 最终配置页面

## 1. 功能概述

用户选择大纲后，进入最终配置页面。本页面让用户设置文章生成的最终参数：
- 是否添加图片
- 是否添加视频
- 是否在文章末尾添加要点总结
- 是否在文章末尾添加FAQ

配置完成后，点击生成按钮，文章开始生成。

## 3. 数据模型

### 3.1 文章配置表

```sql
CREATE TABLE article_configs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    project_id UUID UNIQUE NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
    
    -- 配置选项
    add_images BOOLEAN DEFAULT TRUE,
    add_videos BOOLEAN DEFAULT FALSE,
    add_summary BOOLEAN DEFAULT TRUE,
    add_faq BOOLEAN DEFAULT TRUE,
    
    -- 数量配置
    image_count INTEGER DEFAULT 3,
    faq_count INTEGER DEFAULT 5,
    
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

## 4. API设计

### 4.1 保存文章配置

```
POST /api/projects/{project_id}/article/config
```

**Request Body:**
```json
{
  "add_images": true,
  "add_videos": false,
  "add_summary": true,
  "add_faq": true,
  "image_count": 3,
  "faq_count": 5
}
```

### 4.2 后端实现

```python
# backend/app/api/endpoints/article.py

from fastapi import APIRouter, Depends, HTTPException
from sqlalchemy.ext.asyncio import AsyncSession
from pydantic import BaseModel, Field
from typing import Optional

from app.database import get_db
from app.models.article import ArticleConfig
from app.models.project import Project

router = APIRouter()

class ArticleConfigRequest(BaseModel):
    add_images: bool = Field(default=True)
    add_videos: bool = Field(default=False)
    add_summary: bool = Field(default=True)
    add_faq: bool = Field(default=True)
    image_count: int = Field(default=3, ge=1, le=10)
    faq_count: int = Field(default=5, ge=1, le=10)

@router.post("/projects/{project_id}/article/config")
async def save_article_config(
    project_id: str,
    config: ArticleConfigRequest,
    db: AsyncSession = Depends(get_db)
):
    """保存文章配置"""
    project = await db.get(Project, project_id)
    if not project:
        raise HTTPException(status_code=404)
    
    existing = await db.execute(
        select(ArticleConfig).where(ArticleConfig.project_id == project_id)
    )
    article_config = existing.scalar_one_or_none()
    
    if article_config:
        article_config.add_images = config.add_images
        article_config.add_videos = config.add_videos
        article_config.add_summary = config.add_summary
        article_config.add_faq = config.add_faq
        article_config.image_count = config.image_count
        article_config.faq_count = config.faq_count
    else:
        article_config = ArticleConfig(
            project_id=project_id,
            add_images=config.add_images,
            add_videos=config.add_videos,
            add_summary=config.add_summary,
            add_faq=config.add_faq,
            image_count=config.image_count,
            faq_count=config.faq_count
        )
        db.add(article_config)
    
    project.status = "article_config_saved"
    await db.commit()
    await db.refresh(article_config)
    
    return {"success": True, "config": {...}}
```
